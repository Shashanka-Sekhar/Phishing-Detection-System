## Project Description

This project develops a hybrid AI-based system to detect phishing emails and malicious URLs using transformer-based NLP, domain similarity detection, and behavioral analysis.

## Technologies Used

- Python
- HuggingFace Transformers (BERT)
- PyTorch
- Scikit-learn
- SHAP
- FastAPI
- MySQL
- HTML/CSS/JavaScript
